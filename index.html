      keyboard

      A **keyboard** is an input device used to type text, numbers, and symbols into a computer, smartphone, or other electronic devices. It typically consists of a set of keys arranged in a grid-like pattern. The keys include: - **Alphanumeric keys** (letters and numbers) - **Function keys** (F1, F2, etc.) - **Special keys** (like Enter, Shift, Control, Alt, and the Spacebar) - **Arrow keys** for navigation - **Numeric keypad** (sometimes separate from the main section) When a key is pressed, an electrical signal is sent to the device, allowing the user to input data, execute commands, or interact with software. Keyboards come in different layouts depending on the language and region, with the **QWERTY** layout being the most common.

      Mouse

      A **mouse** is a pointing input device used to interact with a computer or other electronic devices. It allows users to control the on-screen cursor, select objects, and execute commands with precise movements. A typical **mouse** has the following parts: 1. **Buttons**: Usually, two buttons (left and right) are present, and sometimes a scroll wheel in the middle. 2. **Sensor**: This can either be an optical or laser sensor that detects movement on a surface (like a mousepad). 3. **Cable or Wireless**: It can be wired (connected via USB) or wireless (using Bluetooth or a receiver). The mouse allows users to: - **Click**: Select icons, buttons, and files. - **Double-click**: Open programs or files. - **Right-click**: Open context menus for additional options. - **Scroll**: Move up and down through documents or web pages using the scroll wheel. It is a crucial tool for navigation and interaction with graphical interfaces on modern computers.
